Abstract

This research analyzes the impact of Artificial Intelligence on education among young adults aged 15–25 years using Machine Learning techniques. The study focuses on understanding how AI-based educational tools influence academic performance, study habits, and dependency levels. A dataset of 100 student profiles was created containing features such as AI usage hours, study hours, attendance percentage, previous marks, and AI Dependency Index. Linear Regression was implemented to predict student academic performance and evaluate the relationship between AI usage and learning behavior.

AI works best as a supportive educational tool rather than a replacement for self-learning.
